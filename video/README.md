# VMOL Protocol — Demo Video

42-second explainer video built with [Remotion](https://remotion.dev).

## Scenes

1. Title
2. The Problem — governance votes take days, crashes take minutes
3. The Solution — autonomous AI Risk Governor (agent → contract → pool)
4. On-Chain Safety — the 4 guardrail layers
5. Live Result — real Sepolia tx, LTV tightens 75% → 70%
6. Outro

## Commands

```bash
npm install
npm run dev      # open Remotion Studio to preview/edit
npm run render   # render to out/vmol-demo.mp4 (1080p, 30fps)
npm run still    # render a cover image
```

The rendered `out/` folder is gitignored — run `npm run render` to produce the MP4.
